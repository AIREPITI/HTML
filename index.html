DeepSeek API 配置

脚本内置 DeepSeek 大模型接口，需自行获取 API Key 并填入控制面板即可使用。

获取 API Key

前往 DeepSeek 开放平台注册并购买 API 额度，即可获得 API Key

1 点击下方按钮访问 DeepSeek 开放平台
2 注册账户 → 充值余额 → 创建 API Key
3 在脚本悬浮面板 → AI配置 标签页填入 Key 即可

三步安装，即刻使用

只需简单设置即可在你的学习通中体验 AIRE 助手的强大功能。

1

安装脚本管理器

推荐使用 ScriptCat（脚本猫）浏览器扩展，支持 Chrome / Edge / Firefox 等。

2

安装 AIRE 脚本

点击"立即安装"按钮跳转脚本猫安装页面，一键添加脚本到你的脚本管理器中。

3

打开学习通

登录超星学习通平台，进入课程，AIRE 助手将自动启动，悬浮面板出现在页面中。
