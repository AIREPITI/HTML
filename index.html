Installation

三步完成安装

简单几步，即可开始使用

1

安装脚本猫扩展

在浏览器中安装 脚本猫 (ScriptCat) 扩展，支持 Chrome / Edge / Firefox 等主流浏览器。

2

安装脚本

访问 脚本猫安装页面 点击安装，或手动新建脚本粘贴 aire学习通.js 的全部内容后保存。

3

开始使用

打开学习平台课程页面，右上角会出现悬浮面板，开启「自动刷课」和「自动答题」即可开始使用。

AI Setup

配置 AI（可选）

配置后可使用 AI 智能答题功能

1

获取 API Key

前往 DeepSeek 开放平台 注册并获取 API Key。（推荐，便宜好用）

2

填写配置

在悬浮面板「AI 配置」标签页填写 API Key，点击「测试连接」验证配置是否生效。

3

开启 AI 答题

开启「AI 智能答题」开关，脚本将自动使用 AI 分析并回答题目。

FAQ

常见问题

使用前必读

这个脚本是免费的吗？
是的，本脚本完全开源免费（MIT 协议）。AI 答题功能需要自行购买 DeepSeek API Key，费用非常低（约 0.01 元/千 tokens）。
使用脚本会被检测吗？
脚本内置了模拟人工延迟、随机间隔等防检测机制。但请合理使用，建议答题间隔设置为 2-3 秒，避免操作过于频繁。
AI 答题准确吗？
AI 答题基于 DeepSeek V3 大模型，对主观题和复杂题目效果很好。对于客观题（选择/判断），准确率较高。建议答题后人工复核。
支持哪些浏览器？
支持所有基于 Chromium 的浏览器（Chrome、Edge、360 等）以及 Firefox。需要安装脚本猫扩展。
如何更新脚本？
在脚本猫扩展管理页面，找到本脚本，点击「检查更新」即可自动更新到最新版本。
